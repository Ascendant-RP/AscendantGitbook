# ⌨️ Keybinds & Commands

{% hint style="info" %}
Most keybinds can be rebound in **FiveM Settings → Key Bindings → FiveM**.
{% endhint %}

---

## Default Keybinds

### Movement & World

| Key | Action |
|---|---|
| `W` `A` `S` `D` | Move |
| `Shift` | Sprint |
| `Space` | Jump |
| `F` | Enter / exit vehicle |
| `Caps Lock` | Toggle walking style |

### Phone & Inventory

| Key | Action |
|---|---|
| `M` | Open phone |
| `K` | Open inventory |
| `Z` | Check cash on hand |
| `B` | Open job / context menu |

### Combat

| Key | Action |
|---|---|
| `G` | Holster / unholster weapon |
| `Tab` | Open weapon wheel |
| `Left Click` | Fire |
| `Right Click` | Aim |
| `R` | Reload |

### RP Actions

| Key | Action |
|---|---|
| `Y` | Hands up |
| `X` | Hands down |

### Voice

| Key | Action |
|---|---|
| `N` | Cycle voice range — Whisper / Normal / Shout |
| `B` *(radio)* | Push-to-talk on radio channel |

### F-Keys

| Key | Action |
|---|---|
| `F1` | Personal menu |
| `F2` | Inventory shortcut |
| `F3` | Animations quick-menu |
| `F5` | Job menu *(varies by job)* |
| `F8` | FiveM developer console |
| `F10` | ID badge *(LEO / EMS only)* |

## Common Commands

### Roleplay

| Command | Function |
|---|---|
| `/me [action]` | Describe a physical action visible to nearby players |
| `/do [statement]` | Describe something observable or ask an environmental question |
| `/ooc [message]` | Out-of-character chat — use sparingly |
| `/twt [message]` | Post on in-game Twitter/social media |

### Character

| Command | Function |
|---|---|
| `/logout` | Return to character select screen |
| `/cardb` | Open vehicle keys menu |

### Jobs

| Command | Function |
|---|---|
| `/duty` | Toggle on-duty status (LEO / EMS) |
| `/clockin` | Start a job shift |

### Communication

| Command | Function |
|---|---|
| `/phone` | Open phone |
| `/pay [id] [amount]` | Pay another player cash |
| `/givekeys [id]` | Share your vehicle keys |
| `/anim [name]` | Play an animation — see `/anims` for the full list |

### Reporting & Emergency

| Command | Function |
|---|---|
| `/report [description]` | Open a staff ticket in-game |
| `/911 [message]` | Call 911 IC (police and EMS dispatch) |
| `/311 [message]` | Non-emergency police line IC |

{% hint style="warning" %}
Don't spam `/report`. Each use pings online staff. If it's not urgent, open a Discord ticket instead.
{% endhint %}
