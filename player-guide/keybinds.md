# Keybinds & Commands

## Default Keybinds

You can rebind most of these in FiveM Settings → Key Bindings → FiveM.

### Movement & Vehicle
| Key | Action |
|---|---|
| `W A S D` | Move |
| `Shift` | Sprint |
| `Space` | Jump |
| `F` | Enter/exit vehicle |
| `Caps Lock` | Walking style toggle |

### Phone & UI
| Key | Action |
|---|---|
| `M` | Open phone |
| `K` | Inventory |
| `Z` | Check cash on hand |
| `B` | Open menu (job, etc.) |
| `Y` | Hands up |
| `X` | Hands down |

### Combat
| Key | Action |
|---|---|
| `G` | Holster/unholster weapon |
| `Tab` | Weapon wheel |
| `Left Click` | Fire |
| `Right Click` | Aim |
| `R` | Reload |

### Voice
| Key | Action |
|---|---|
| `N` | Cycle voice range (whisper / normal / shout) |
| `B` (radio) | Push-to-talk on radio |

## Common Commands

### Communication
- `/me [action]` — describe a physical action
- `/do [statement]` — describe an environmental detail or ask a question
- `/ooc [message]` — out-of-character chat (use sparingly)
- `/twt [message]` — post on Twitter (in-game)

### Phone
- `/phone` — open phone
- `/togglephone` — show/hide phone in hand

### Character
- `/logout` — return to character select
- `/cardb` — open vehicle keys menu

### Jobs
- `/duty` — toggle duty (LEO/EMS)
- `/clockin` — start a job

### Reporting
- `/report [issue]` — open a staff ticket
- `/911 [emergency]` — call 911 IC
- `/311 [non-emergency]` — non-emergency police IC

### Misc
- `/pay [id] [amount]` — pay another player cash
- `/givekeys [id]` — share vehicle keys
- `/anim [name]` — play an animation (full list: `/anims`)

## F-Keys

| Key | Action |
|---|---|
| `F1` | Personal menu |
| `F2` | Inventory shortcut |
| `F3` | Animations menu |
| `F5` | Job menu (varies) |
| `F8` | FiveM console |
| `F10` | ID badge (LEO/EMS) |
