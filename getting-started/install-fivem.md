# ⬇️ Installing FiveM

FiveM is the free modification framework that lets you connect to custom GTA V multiplayer servers like Ascendant RP.

---

## Requirements

{% hint style="warning" %}
You **must** own a legitimate copy of GTA V (Steam, Epic, or Rockstar Launcher). FiveM does not work with pirated copies.
{% endhint %}

- ✅ Legitimate copy of **GTA V**
- ✅ **Windows 10** or newer
- ✅ A working **microphone** — Ascendant is a voice-RP server
- ✅ A **Cfx.re account** (free — linked to your Discord)

---

## Installation Steps

1. Go to [fivem.net](https://fivem.net/) and click **Download Client**.
2. Run `FiveM.exe` — do **not** install it inside `Program Files`.
3. On first launch, FiveM will detect your GTA V installation. If it can't find it automatically, point it to your `GTA5.exe`.
4. Sign in with a **Cfx.re account**. Create one at [cfx.re](https://cfx.re) if you don't have one.
5. Link your **Discord account** to your Cfx.re profile — this is required to join Ascendant.

---

## Recommended Settings

| Setting | Recommendation |
|---|---|
| **Streaming Memory** | 8 GB+ (Settings → Game) |
| **Cache** | Clear every few weeks (`AppData\Local\FiveM\FiveM Application Data\data\cache`) |
| **Voice Input** | Set your mic under FiveM Settings → Game |

---

## Common Install Issues

| Problem | Fix |
|---|---|
| "GTA V not found" | Manually browse to your `GTA5.exe` |
| Stuck on splash screen | Run FiveM as Administrator |
| Crashes on join | Clear FiveM cache, then verify GTA V files via Steam/Epic |
| Can't hear other players | Check Windows audio device settings + FiveM voice settings |
| Black screen on launch | Update your GPU drivers |
