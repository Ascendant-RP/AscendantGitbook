# Installing FiveM

FiveM is the modification framework that lets you connect to custom GTA V multiplayer servers like Ascendant.

## Requirements

- A **legitimate copy of GTA V** (Steam, Epic, or Rockstar Launcher).
- Windows 10 or newer.
- A working microphone — we are a voice-RP server.

## Installation Steps

1. Go to [https://fivem.net/](https://fivem.net/) and click **Download Client**.
2. Run `FiveM.exe` and let it install to its own folder (do **not** put it in `Program Files`).
3. On first launch, FiveM will detect your GTA V install. If it can't find it, point it at your `GTA5.exe`.
4. Sign in with a Cfx.re account (create one if you don't have it). Linking your **Discord** is required to join Ascendant.

## Recommended Settings

- **Streaming memory:** 8 GB or higher if you have it (Settings → Game).
- **Cache:** clear your FiveM cache every couple of weeks (`AppData\Local\FiveM\FiveM Application Data\data\cache`).
- **Voice:** set your input device under FiveM Settings → Game.

## Common Install Issues

| Problem | Fix |
|---|---|
| "GTA V not found" | Manually browse to your `GTA5.exe`. |
| Stuck on splash screen | Run FiveM as Administrator. |
| Crashes on join | Clear cache, verify GTA V files via Steam/Epic. |
| Can't hear voices | Check Windows audio device + FiveM voice settings. |
