# 🔗 How to Connect

{% hint style="warning" %}
**Whitelist required.** If you haven't been approved yet, [apply here](whitelist.md) before trying to connect.
{% endhint %}

---

## Method 1 — Direct Connect (Recommended)

1. Open **FiveM**.
2. Press **F8** to open the console.
3. Type the command below and press **Enter**:

```
connect cfx.re/join/xlzgp5m
```

4. Wait for assets to download. Your **first connect** may take 5–15 minutes.

---

## Method 2 — Server Browser

1. Open FiveM → click **Play** → **Servers**.
2. Search for **Ascendant RP**.
3. Click the server and hit **Connect**.

---

## Method 3 — Cfx.re Link

Click the link below — FiveM will open and begin connecting automatically:

👉 **[cfx.re/join/xlzgp5m](https://cfx.re/join/xlzgp5m)**

---

## First-Time Connect

The first time you join, FiveM will download all custom assets:

- Custom vehicle models and textures
- Map MLOs (custom interiors)
- UI resources and scripts

{% hint style="info" %}
**Don't disconnect mid-download.** If you do, you'll have to start the download over. Let it fully load even if it appears stuck for a minute.
{% endhint %}

---

## Connection Issues

| Problem | Fix |
|---|---|
| "You are not whitelisted" | Your application is pending or was denied — check [#whitelist-status](https://discord.gg/ascendantrp) on Discord |
| Stuck on "Awaiting Scripts" | Wait — large-resource servers can take 60+ seconds to fully initialize |
| "Connection failed" | Verify your Discord is linked to your Cfx.re account at [cfx.re](https://cfx.re) |
| Crash during loading screen | Clear FiveM cache, then retry |
| Disconnected immediately | Make sure you're running the latest version of FiveM |

{% hint style="info" %}
If you're stuck for more than 10 minutes, ask in [#tech-support](https://discord.gg/ascendantrp) on Discord with a screenshot of your error.
{% endhint %}
