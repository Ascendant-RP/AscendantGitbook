# Getting Started

Welcome aboard! This section will get you from "I just heard about Ascendant" to "I'm spawned in Los Santos and ready to RP."

## In This Section

- [About the Server](about.md) — what makes Ascendant different
- [Installing FiveM](install-fivem.md) — the launcher you'll need
- [How to Connect](how-to-connect.md) — joining the server
- [Whitelist Application](whitelist.md) — how to get approved

## TL;DR

1. Install **FiveM** (free, requires legitimate GTA V).
2. Join our [Discord]([DISCORD_INVITE_URL]) and read the rules.
3. Submit a [whitelist application](whitelist.md).
4. Once approved, connect to `[SERVER_CONNECT_URL]` or search **Ascendant RP** in the FiveM server browser.
5. Create your character and start your story.
