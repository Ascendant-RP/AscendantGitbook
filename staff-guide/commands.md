# 🔧 Admin Commands

{% hint style="warning" %}
Full command syntax, permission tiers, and keybinds are in the **staff-only Discord pinned document**. This page is a public-facing summary so the community understands what tools staff have access to.
{% endhint %}

---

## Command Categories

### 🔍 Player Lookup
- View a player's character info, current location, and inventory.
- Check ban and punishment history.
- Review recent chat and combat logs for any player.

### 🧭 Movement & Spectating
- `noclip` — move freely through the world. **Logged.**
- `tp [player/coords]` — teleport to a player or set of coordinates. **Logged.**
- `bring [player]` — bring a player to your location. Requires consent unless it's a rescue. **Logged.**
- `spectate [player]` — invisibly observe a player's perspective. **Logged.**

### ⚖️ Moderation Actions
| Command | Function |
|---|---|
| `kick [id] [reason]` | Remove player from server; they can rejoin |
| `ban [id] [duration] [reason]` | Apply a temp or permanent ban |
| `warn [id] [reason]` | Issue a logged warning |
| `mute [id] [duration]` | Mute player in voice or text |
| `freeze [id]` | Freeze a player's movement |
| `unfreeze [id]` | Release a frozen player |

### 💰 Compensation (Restricted)
Item, vehicle, or money compensation requires:
- **Head Admin approval**
- A documented ticket with video evidence
- Double sign-off logged in the record

### 🖥️ Server Management (Senior Admin+)
- Restart vote
- Hot-reload a resource
- Kick all players
- Server-wide broadcast

---

## All Commands Are Logged

Every admin command produces a log entry containing:

- The acting staff member
- The target player
- Command + arguments used
- Timestamp
- Reason (where required)

Logs are reviewed during **monthly audits**. Command misuse is grounds for demotion or removal.

---

## What Admin Tools Are NOT For

{% hint style="danger" %}
Using admin tools outside of ticket work is an abuse of power and will result in disciplinary action.
{% endhint %}

- ❌ Bypassing IC consequences for yourself or your character
- ❌ Looking up a friend's location or inventory "for fun"
- ❌ Spectating players for entertainment, not ticket investigation
- ❌ Running lookups without an active ticket reason
