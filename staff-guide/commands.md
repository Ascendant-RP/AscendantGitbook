# Admin Commands

> Detailed command syntax and permission tiers are maintained in the **staff-only Discord pinned doc**. This page is a public-facing summary so players know what tools exist.

## Categories

### Player Lookup
- Look up a player's character info, location, and inventory.
- Check ban/punishment history.
- View recent chat and combat logs for a player.

### Movement & Spectating
- `noclip` — move freely through the world (logged).
- `tp` to a player or coordinates (logged).
- `bring` a player to your location (with consent for non-rescue).
- `spectate` a player (logged, invisible to them).

### Moderation Actions
- `kick [id] [reason]`
- `ban [id] [duration] [reason]`
- `warn [id] [reason]`
- `mute [id] [duration] [reason]`
- `freeze` / `unfreeze [id]`

### Compensation (Restricted)
- Item / vehicle / money compensation requires Head Admin approval and a documented ticket reason. Logged with double sign-off.

### Server Management (Senior+)
- Restart vote, hot-reload resource, kick all, broadcast.

## Logging

Every admin command produces a log entry containing:
- Acting staff member
- Target player
- Command + arguments
- Timestamp
- Reason (if required)

Logs are reviewed during monthly audits. Misuse is grounds for demotion or removal.

## Don't Use Admin Tools For

- Bypassing IC consequences for yourself or your character
- Looking up "your friend's address" IC
- Spectating specific players for entertainment
- "Just to check" something without a ticket reason
