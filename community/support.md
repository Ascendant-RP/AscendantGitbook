# Support

Stuck? Here's how to get help, fast.

## Quick Routing

| I have a... | Go to |
|---|---|
| **Tech issue** (can't connect, crash, install) | `#tech-support` on [Discord]([DISCORD_INVITE_URL]) |
| **Bug to report** | `#bug-reports` |
| **Question about a rule** | `#support-tickets` |
| **Player to report** | In-game `/report` or `#support-tickets` |
| **Compensation request** | In-game `/report` with evidence |
| **Ban to appeal** | `#ban-appeals` |
| **Donation issue** | `#support-tickets`, mark as **Billing** |

## Opening an In-Game Ticket

While connected, type:
```
/report [brief description of the issue]
```

A staff member will reach out as soon as one is available. **Don't spam** the command — it pings staff each time.

## Opening a Discord Ticket

1. Go to `#support-tickets`.
2. Click the appropriate category button.
3. Fill the prompts.
4. Wait for a staff response (usually within a few hours; longer overnight).

## What to Include

The more info you give upfront, the faster we can help:

- **Your character name + Cfx ID**
- **What happened** (timeline, who was involved)
- **Evidence** — clips, screenshots, video links
- **What you've already tried** (for tech issues)

## Response Times

| Type | Typical Response |
|---|---|
| In-game ticket | < 15 min during peak hours |
| Discord support ticket | < 6 hours |
| Bug report | Triaged daily |
| Ban appeal | 5–7 days |
| Whitelist application | Up to 48 hours |

## What's Not Supported

- "Can I have free money / a car?" — no.
- "I lost something doing something I shouldn't have" — no comp for self-inflicted issues.
- "I don't like this rule, change it" — open a `#suggestions` post instead.

## Emergency Contact

If something serious is happening (active raid, doxxing, threats), ping `@Staff` in `#support-tickets`. Use this **only** for emergencies.
